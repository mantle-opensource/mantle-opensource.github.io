Mantle

Efficient Hierarchical Metadata Management for Cloud Object Storage Services

The Mantle paper presented at SOSP'25 introduces a novel hierarchical namespace metadata management approach for cloud object storage services. Mantle has been deployed on Baidu Object Storage (BOS) for over 2 years, a service developed by Baidu Canghai Storage. We will open-source the prototype implementation to support further research and collaboration.
