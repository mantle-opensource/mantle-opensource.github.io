Mantle: Efficient Hierarchical Metadata Management for Cloud Object Storage Services

The Mantle paper presented at SOSP'25 introduces a novel hierarchical namespace metadata management approach for cloud object storage services. We will open-source the prototype implementation to support further research and collaboration.
